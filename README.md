## Issues Faced and Solutions

### Finding Extensions in Cursor
Initially, I was unsure where to find the Extensions panel in Cursor. I learned that the shortcut `Ctrl + Shift + X` opens it directly.

### GitHub Authorization
During setup, GitHub requested authorization permissions. After reviewing the permissions, I successfully connected GitHub with Cursor.

### Repository Setup
I initially had trouble understanding how to create and open a repository, but after creating a public repository with a README file, the process became clear.
